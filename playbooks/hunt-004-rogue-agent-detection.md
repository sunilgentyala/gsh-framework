# Hunt-004: Rogue Agent Detection

**Status:** DRAFT — Coming Q2 2026
**Threat Class:** Rogue Agent / Unauthorized Tool Use
**Severity:** Critical
**MITRE ATLAS:** AML.T0053
**NIST CSF 2.0:** PR.PS-04, RS.AN-03

## Overview

This playbook covers detection of unauthorized or rogue agents operating
within multi-agent pipelines, including agents that exceed their declared
tool permissions, impersonate trusted agents, or establish covert
communication channels outside sanctioned orchestration flows.

## Status

Full playbook in development. See Hunt-001 through Hunt-003 for
currently available detection playbooks.
